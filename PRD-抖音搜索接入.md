# PRD：抖音搜索接入方案

> 版本：v0.1 | 日期：2026-05-17

---

## 1. 现实约束

抖音**没有对个人开发者开放搜索 API**。官方开放平台 (open.douyin.com) 的接口需要企业认证 + 特定权限，且不提供视频文案搜索。

所以不是在"用哪个 API"之间选，而是在**不同实现路径**之间选。

---

## 2. 三条可行路径

### 路径 A：第三方数据平台 API（推荐）

国内有几家专门做短视频数据的平台，提供付费/免费 API：

| 平台 | 能查什么 | 价格 |
|------|---------|------|
| **蝉妈妈** (chanmama.com) | 视频文案搜索、达人搜索、热点榜单 | 付费，有试用 |
| **飞瓜数据** (feigua.cn) | 同类型视频聚合、文案分析、话题热榜 | 付费 |
| **新榜** (newrank.cn) | 跨平台内容搜索、爆款排行、话题趋势 | 部分免费 |
| **卡思数据** (caasdata.com) | 达人分析、视频搜索 | 付费 |
| **巨量算数** (trendinsight.oceanengine.com) | 抖音官方数据平台，话题趋势、搜索热词 | 免费 |

**接入方式：**

```
CopyAgent
    │
    ├─ 搜索卖点/痛点时 → 调用 DeepSeek（已实现）
    │
    └─ 搜索抖音爆款时 → 调用第三方 API
                          │
                          ├─ 新榜 API（免费额度，搜爆款标题和文案）
                          └─ 蝉妈妈 API（付费，搜索结果更全）
```

**优点：** 合法、稳定、数据结构化
**缺点：** 部分付费、调用频率有限

### 路径 B：网页搜索 + DeepSeek 分析（零成本，立即可用）

用通用搜索引擎搜"抖音 爆款 [产品名]"，把搜索结果喂给 DeepSeek 提取文案：

```
用户输入产品："加热棒"
        │
        ▼
1. 调用 Bing/Google 搜索 "抖音 加热棒 爆款文案"
        │
        ▼
2. 取前 10 条搜索结果（标题 + 摘要）
        │
        ▼
3. DeepSeek 从结果中提取视频文案片段、总结常用的表达方式
        │
        ▼
4. 输出：该产品在抖音上常见的钩子类型 + 金句 + 卖点角度
```

**优点：** 免费、无需注册第三方、立即可用
**缺点：** 拿不到完整视频文案，只能拿到片段和归纳

### 路径 C：手动 + 半自动（已有功能）

用户自己刷抖音看到好的 → 复制文案 → 粘贴到 CopyAgent 的「爆款拆解」：

```
已实现：用户粘贴文案 → DeepSeek 拆解 → 存入爆款库 → 仿写时引用
待增强：支持粘贴抖音分享链接，系统自动提取文案
```

链接提取需要用 headless browser（Playwright/Selenium）打开抖音分享链接，抓取页面中的视频描述。**但这个做法不稳定、可能被封。**

## 3. 推荐实施路线

```
Phase 1（立即可做，0 成本）
  └─ 接入 Bing Search API（免费额度 1000次/月）
     → 搜索 "抖音 爆款 [产品]" → DeepSeek 提取精华 → 作为卖点来源之一

Phase 2（月费约 100-300 元）
  └─ 接入新榜 API
     → 按产品/关键词搜索抖音爆款 → 获取真实文案 → 直接拆解

Phase 3（月费约 500-1000 元，数据最全）
  └─ 接入蝉妈妈 API
     → 搜视频文案 + 达人 + 趋势 + 话题标签
```

### 4.2 Bing Search API

微软 Bing Web Search API 有免费层：每月 1000 次调用，足够个人用。

注册地址：https://www.microsoft.com/en-us/bing/apis/bing-web-search-api

在 CopyAgent 中新增一个文件 `search/douyin_search.py`：

```python
# 伪代码
def search_douyin_trends(product: str) -> dict:
    """搜索产品在抖音上的爆款文案趋势"""
    
    # 1. Bing 搜索
    queries = [
        f"抖音 爆款 {product} 文案",
        f"{product} 口播 文案 抖音",
        f"{product} 卖货 话术 抖音",
    ]
    search_results = []
    for q in queries:
        results = bing_search(q, count=10)
        search_results.extend(results)
    
    # 2. DeepSeek 提取
    prompt = f"""
    以下是关于「{product}」在抖音上的搜索结果摘要。
    请提取其中提到的：
    1. 视频文案片段（如果有完整文案，直接摘录）
    2. 常用的开头钩子类型
    3. 被反复使用的卖点角度
    4. 高频关键词和话术
    
    搜索结果：
    {search_results}
    """
    
    return deepseek_analyze(prompt)
```

## 5. 推荐先做哪个？

| 方案 | 成本 | 时间 | 能拿到什么 |
|------|------|------|-----------|
| **Bing + DeepSeek** | 免费 | 半天 | 抖音文案的片段和趋势归纳 |
| 新榜 API | 付费 | 半天 | 真实爆款标题 + 部分文案 |
| 蝉妈妈 API | 较贵 | 半天 | 完整文案 + 达人数据 |

**建议先做 Bing + DeepSeek**，验证需求强度。如果日常够用就省钱了，不够用再接入付费 API。
